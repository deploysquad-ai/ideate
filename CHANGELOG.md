# Changelog

## [0.1.0] - Unreleased

### Added
- Initial plugin structure
- `/ideate` — session orchestration with conversational intent detection
- `/ideate.branch` — branch creation and switching
- `/ideate.merge` — squash-merge and abandon
- `/ideate.doc` — structured document generation
- `/ideate.research` — web research scoped to current thread
- Session persistence in `.ideate/` directory
- Six artifact types: feature, decision, constraint, persona, goal, module
